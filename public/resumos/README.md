# Imagens dos Resumos

Esta pasta deve conter as imagens dos resumos de cada matéria.

## Arquivos necessários:

- `biologia.jpg` - Imagem do resumo de Biologia
- `fisica.jpg` - Imagem do resumo de Física
- `quimica.jpg` - Imagem do resumo de Química
- `geografia.jpg` - Imagem do resumo de Geografia
- `historia.jpg` - Imagem do resumo de História
- `literatura.jpg` - Imagem do resumo de Literatura
- `portugues.jpg` - Imagem do resumo de Português
- `matematica.jpg` - Imagem do resumo de Matemática
- `sociologia.jpg` - Imagem do resumo de Sociologia
- `filosofia.jpg` - Imagem do resumo de Filosofia

## Formato recomendado:

- **Formato**: JPG ou PNG
- **Dimensões**: 800x600px ou proporção similar
- **Tamanho**: Máximo 500KB por imagem (para melhor performance)
- **Qualidade**: Boa qualidade visual, legível

## Como adicionar:

1. Coloque as imagens nesta pasta (`public/resumos/`)
2. Use os nomes exatos listados acima (em minúsculas)
3. As imagens serão exibidas automaticamente nos cards das matérias
4. Se uma imagem não existir, será exibido um ícone padrão como fallback
